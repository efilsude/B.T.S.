# Widgets Klasörü

## Amacı
Tekrar kullanılabilir UI bileşenlerini barındırır.

## İçerik
- `custom_button.dart` - Özelleştirilmiş buton
- `custom_text_field.dart` - Özelleştirilmiş text field
- `photo_picker.dart` - Fotoğraf seçici widget
- `info_grid.dart` - Bilgi grid bileşeni
- `photo_grid.dart` - Fotoğraf galerisi bileşeni

## Kullanım
Widget'lar, ekranlar arasında tekrar kullanılabilir UI parçalarıdır.
