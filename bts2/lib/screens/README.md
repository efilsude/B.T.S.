# Screens Klasörü

## Amacı
Uygulamadaki tam ekran sayfaları (pages/views) barındırır.

## İçerik
- `home_screen.dart` - Ana sayfa
- `cover_page.dart` - Rapor kapak sayfası
- `report_form.dart` - Rapor formu
- `report_list.dart` - Rapor listesi
- `pdf_preview.dart` - PDF önizleme

## Kullanım
Her ekran, Navigator ile yönlendirilir ve tam sayfa görüntüler.
