# Models Klasörü

## Amacı
Uygulamadaki veri modellerini (data classes) barındırır.

## İçerik
- `trafo_report.dart` - Trafo bakım raporu veri modeli
- `islem_turu.dart` - İşlem türü enum'ı
- Diğer entity modelleri

## Kullanım
Veri modelleri, uygulama genelinde veri taşımak ve API/local storage ile iletişim için kullanılır.
