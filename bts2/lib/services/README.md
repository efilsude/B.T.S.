# Services Klasörü

## Amacı
Harici servis ve iş mantığını barındırır.

## İçerik
- `pdf_service.dart` - PDF oluşturma servisi
- `storage_service.dart` - Yerel depolama (Hive/SQLite) servisi
- `photo_service.dart` - Fotoğraf işleme servisi

## Kullanım
Service sınıfları, dış sistemlerle (dosya sistemi, API, database) iletişim kurar.
